GRIP - Vision Processing

The WPIRoboticsProjects GRIP project. GRIP is a "program for rapidly developing computer vision applications" used by many FRC Teams for vision tracking and other vision-related projects.

Controller Wrappers

Java FRC Wrappers for various controller devices, including  the XBox360, Logitech Attack 3, Logitech Extreme 3D, and more.

WinCommander AHK

WinCommander is a short and simple AutoHotkey script that allows you to toggle keeping a window "Always-Ontop", open a window or media in a small borderless window, and auto-position and auto-scale them to the bottom-right corner.

AutoHotkey

FRC Dashboard

A custom HTML JS Network Table based dashboard builder, allowing sleeker designs and a more robust level of control. However, as of 2018, Shuffleboard has been released for building JavaFX dashboards in Java.
